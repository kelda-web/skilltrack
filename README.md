# SkillTrack — Skill Assessment & Learning Management System

A full-stack LMS where users can choose skills, follow learning roadmaps, track progress, and take assessments.

## Tech Stack
- Frontend: React.js, Vite, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MySQL, Sequelize
- Auth: JWT
- Testing: Postman (manual API testing)

## Features
- User authentication (JWT-based)
- Skill selection & personalized learning roadmaps
- Progress tracking dashboard
- Assessment module

## Setup

**Backend**
```bash
cd backend
npm install
npm start
```

**Frontend**
```bash
cd frontend
npm install
npm run dev
```
